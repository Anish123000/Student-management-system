# Student Management System (C++)

## 📌 Overview

This is a simple console-based Student Management System that I built using C++ to practice core programming concepts.
The project helps manage student records like adding, updating, searching, and deleting data in an efficient way.

## 🚀 Features

* Add new students
* Display all student records
* Search student by ID
* Update student marks
* Delete student record
* Find the topper
* Calculate average marks
* Prevent duplicate student IDs

## 🛠️ Technologies Used

* C++
* Arrays
* Structures

## ▶️ How to Run

1. Compile the code:
   g++ main.cpp -o main
2. Run the program:
   ./main
